# Orbit Orchard

A fast, colorful React + TypeScript arcade game built with Vite.

Tune the comet core to a color, harvest matching orbiting seeds, and keep the orchard glow alive for as long as possible. Correct matches build combo score and restore energy. Wrong harvests drain the orchard.

## Play

```bash
npm install
npm run dev
```

Then open:

```text
http://127.0.0.1:5173
```

## Controls

- Click or tap orbiting seeds to harvest them.
- Click a color swatch to tune the comet core.
- Press `1`, `2`, `3`, or `4` to switch colors.
- Press `Space` to pause or resume.
- Press `R` to reset.
